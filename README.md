# NextJs Todo App

> NextJs simple todo app

## Features

-Typescript
-TailwindCSS
-Prisma
-Winston logger

### Instalation

```bash
# Install dependencies
npm run install

# .env
cp .env.example .env

# Change username, password, and db : DATABASE_URL="mysql://username:password@localhost:3306/db"

# Migrate db
npx prisma migrate
```

### Development
```bash
npm run dev
```
